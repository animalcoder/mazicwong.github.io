数据库之SQL Server

According to the follow official document, this is my first attempt to learn the ‘SQL Server’:

https://docs.microsoft.com/en-us/sql/linux/quickstart-install-connect-ubuntu

  • SQL Server

    • 2. install the SQL Server command-line tools: sqlcmd AND bcp
