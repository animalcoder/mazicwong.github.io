卡特兰数

卡特兰数

http://www.cnblogs.com/kuangbin/archive/2012/03/21/2410516.html

hdu 1023 卡特兰数+大数

递推关系：
当h(1)=h(2)=1:
h(n) = h(1)h(n-1)+h(2)h(n-2)+…+h(n-1)h(n)
当h(1)=1,h(2)=2:
h(n) = h(n-1)(4n-2)/(n+1)
即h(n) = C(2n,n)/(n+1)

应用：

  1. 出栈次序
    n个数顺序进栈，问出栈顺序
  2. 二叉树构成问题
    n个节点，问能构成多少种二叉树
  3. 凸多边形的三角形划分
    一个凸n边形划分成多个三角形，有几种分发
  4. 矩阵链乘
    括号匹配问题
  5. 其他
    01序列问题、n边形格子从左上角到右下角不跨过对角线问题
