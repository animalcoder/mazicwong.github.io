In ubtuntu

本来想在服务器上面试试的，不过那边好像出了点问题，直接装本机上了
java环境之前就有了，主要是装下mysql，不过照着书上做的时候出了点问题

  • 安装
    sudo apt-get install mysql-client mysql-service libmysql-java

  • 问题
    运行过程一直出现”No Suitable Driver Found For Jdbc“，

  • 解决:
    添加

    1. 驱动加入路径中,bashrc
      CLASSPATH=$CLASSPATH:/usr/share/java/mysql.jar
    2. 把jar包放到对应jre的外部jar包中！
      sudo cp /usr/share/java/mysql-connector-java-5.1.38.jar /usr/lib/jvm/java-8-openjdk-amd64/jre/lib/ext

  • 参考:

    http://www.blogjava.net/w2gavin/articles/217864.html

  • 关闭开机启动,不然关机忘了关mysql就很慢了= =
    注释掉/etc/init/mysql.conf的#start on runlevel [2345]
